## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

This project is a clone of the popular Wordle game, where players attempt to guess a 5-letter word within six tries. Each guess provides feedback on how close the letters are to the correct word. 

Here's why this project is useful:
* Practice your coding skills by building a fully functional game
* Learn how to use React for state management and user interactions
* Implement a light/dark mode toggle for a better user experience

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/wordle-clone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

3. Change git remote url to avoid accidental pushes to base project
   ```sh
   git remote set-url origin github_username/repo_name
   git remote -v # confirm the changes
   ```
